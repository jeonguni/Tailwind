Tailwind CSS 설치

npm install -D tailwindcss
npx tailwindcss init

터미널에 cd tailwind 설치할 폴더 드래그 후, 위에 코드 한줄씩 입력
그럼 tailwind.config.js 파일 생성됨

content: ["./src/**/*.{html,js}"],

해당 경로 입력하기

src/파일명.css 만들기

기본 css만들기

@tailwind base;
@tailwind components;
@tailwind utilities;

파일명.css에 위 코드 import하기

npx tailwindcss -i ./src/파일명.css -o ./dist/style.css --watch

터미널로 돌아와서 코드 입력

link rel="stylesheet" href="dist/style.css"
